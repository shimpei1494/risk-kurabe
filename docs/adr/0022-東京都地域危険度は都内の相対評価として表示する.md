# ADR-0022: 東京都地域危険度は都内の相対評価として表示する

- 状態: 採用
- 決定日: 2026-07-17

## コンテキスト

東京都の「地震に関する地域危険度測定調査」に含まれる建物倒壊危険度と火災危険度は、東京都内の町丁目を相対的に評価した指標である。ランクを絶対的な被災確率や安全性として表示すると、指標の意味を超えた解釈になる。

ハッカソン版は関東1都6県を対象とするが、この地域危険度は東京都だけの地域追加リスク情報である。東京都外の地点を最低ランクとして扱うと、データがないことを低リスクと誤認させる。

## 決定

- 建物倒壊危険度と火災危険度は別々の指標として表示する。
- 主表示には公式のランク1〜5を使用する。
- ランクには`東京都内における相対評価`であることを併記する。
- ランク1を`安全`、ランク5を絶対的な`危険`または被災確率として表現しない。
- 元データに危険量などの根拠数値が含まれる場合は、単位と出典を付けて詳細欄に表示する。
- 東京都外の地点はランクへ変換せず、データ状態を`対象外`とする。
- 東京都内と東京都外の地点を比較するとき、この指標による地点間の優劣または順位を作らない。
- AI要約は、公式ランク、都内の相対評価であることおよび必要な根拠情報だけを説明し、被災確率や絶対的安全性へ読み替えない。

## 表示要件

- 比較表では、東京都外のセルに`対象外（東京都のみの指標）`と表示する。
- 地図の凡例には、ランクの方向とともに`東京都内の町丁目間の相対評価`と表示する。
- ランクの色だけに意味を依存せず、数値とテキストを常に併記する。
- 詳細欄から調査名、調査回、基準時点および東京都の出典ページを確認できるようにする。

## 結果

- 東京都外のデータ欠如を低リスクとして扱う誤比較を防げる。
- ランクが示す範囲を超えた安全・危険の断定を避けられる。
- 東京都内の地点同士では同じ公式指標による事実比較が可能になる。
- 関東全域で共通する地震リスク比較は提供できず、将来別の広域データを導入するまで地域差が残る。
