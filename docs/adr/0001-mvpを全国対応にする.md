# ADR-0001: MVPを全国対応にする

- 状態: ADR-0016により置換
- 決定日: 2026-07-17

## コンテキスト

事前定義では、全国共通の洪水データと東京都独自の地震危険度データを扱う一方、主なターゲットユーザーを東京都への、または東京都内での引越し検討者としていた。このままでは、MVPが受け付ける比較地点の地理的範囲が曖昧になる。

将来的には、全国で利用できるデータについては東京都外の地点も比較できるサービスを目指している。

## 決定

MVPから日本全国の地点を比較対象にする。

- 全国の比較地点では、全国共通リスク情報を取得・表示する。
- 東京都内の比較地点では、全国共通リスク情報に地域追加リスク情報を加える。
- 東京都外の比較地点では、東京都限定データを `対象外` と表示する。
- 地域追加リスク情報を後から他地域へ拡張できるよう、全国共通情報と地域追加情報を分離して設計する。

## 置換

2週間・1人のハッカソン開発という制約と、頻度別浸水データの処理量を踏まえ、ハッカソン版の対象地域はADR-0016で関東1都6県へ変更した。全国対応は継続運用フェーズの目標として残す。

## 結果

- 東京都外から東京都への引越しを含め、全国の任意の地点を同じ比較に含められる。
- 住所検索、地図表示および全国共通データの処理は、MVPから全国を対象にする必要がある。
- 地点ごとに利用可能な情報が異なるため、値だけでなくデータ状態と根拠情報も比較結果の一部として扱う必要がある。
- MVPの開発量は東京限定の場合より増えるが、全国展開のための地理スコープ変更は不要になる。

## 未決事項

- 住所検索および座標確定に使うサービス
- 全国共通データの取得・前処理・配信方式
- 地域追加リスク情報を適用するための地域判定方式
- 比較結果を保持・共有する範囲
